# Configuration
Ce projet utilise le Runtime [.NET Framework version 4.5](https://www.microsoft.com/en-us/download/details.aspx?id=30653) et a été développé sous Windows 10. Le projet devrait être rétrocompatible avec Windows 11, mais je ne peux pas le garantir.

# Décharge
Ce projet est laissé en l'état. En utilisant ce projet, vous comprenez que ce logiciel n'est plus supporté depuis longtemps et que donc, aucun bug ni faille de sécurité ne sera corrigée.
